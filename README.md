# AAO Developer Docs

Documentation for contributors to the [All Aboard Ohio Developer Program](https://github.com/all-aboard-ohio).

Each `.md` file in this repository is automatically surfaced in the Dev Docs section of the AAO developer homepage. To add a new doc, create a new `.md` file — frontmatter is optional but recommended.

## Frontmatter format

```yaml
---
title: Your Doc Title
excerpt: A one-sentence description shown in the card preview.
---
```
